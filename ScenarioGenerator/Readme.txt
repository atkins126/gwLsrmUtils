Scenario Generator 
предназначен для создания больших сценариев по шаблону, который будет расширен на список спектров.

Template file задаётся сценарий-шаблон, который может содержать следующие переменные, которые будут заполнены при генерации:
[SPE_PATH] -- полный путь к файлу спектра, например, "c:\Lsrm\Work\GP\HPGe(20%)\Spe\Point25\Am241-6649-25cm.spe"
[SPE_NAME] -- имя файла спектра без расширения, например, "Am241-6649-25cm"
[PARENT_DIR] -- родительская директория для спектра, например, Point25

Spe directory -- директория, содержащая спектры для которых будет генериться сценарий. Спектры добавляются рекурсивно.

Mask for spe -- маска для файлов спектра, например, "*.spe" -- будут добавляться в сценарий только spe-файлы

Пример шаблонного сценария:
open([SPE_PATH])
runscenario(U_48-258keV_silent.lsc)
output(Results\48-258\Result_[SPE_NAME].txt, 1)
close

для всех спектров в Spe directory будет по очереди открываться спектр, запускаться сценарий U_48-258keV_silent.lsc, выводиться результаты в файл Results\48-258\Result_[SPE_NAME].txt, где [SPE_NAME] будет заменено на имя файла и закрываться окно спектра
