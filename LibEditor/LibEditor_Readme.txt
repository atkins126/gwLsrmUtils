Программа LibEditor предназначена для открытия и сохранения библиотек нуклидов в форматах lsrm (lib), текстовом формате, раделённых табом (txt) и коротких текстовых библиотек с только периодами полураспада в годах. Программа конверитирует из одного формата в другой.
Возможности программы:
- Программа позволяет сортировать список нуклидов по имени (sort by name) и по массовому числу (sort by mass number).
- позволяет сортировать энергии одного нуклида (ПКМ по списку энергий нуклида) и делать сортировку энергий по всем  нуклидам (ПКМ по списку энергий нуклида или из меню Tools)
- позволяет объединять несколько нуклидов в один (выделить нуклиды в списке нуклидов с shiftом или ЛКМ, потом по ПКМ выбрать Merge Nuclides). Предназначена для объединения дочерних нуклидов в равновесный
- позволяет конверитировать русские единицы периода полураспада в английские
- позволяет открыть библиотеку путь - 1-й параметр командной строки. Программа запускается с загруженной библиотекой. Если указан 2-й параметр, то сохранить в другом формате путь - 2-й параметр командной строки, при этом программа автоматически закрывается.
- позволяет рассчитывать мощности дозы из активности (по кнопке Calculate dose power)
Параметры командной строки:
LibEditor.exe [<OpenLibName> [SaveLibName]]
	<OpenLibName> - имя библиотеки, которая загрузится в программу при её запуске. Расширение lib - формат LSRM, остальные расширения -- текстовый формат, разделённые табом.
	<SaveLibName> - имя библиотеки, в которую произойдёт автоматическая конвертация и сохранение при запуске программы. Расширение lib - формат LSRM, остальные расширения -- текстовый формат, разделённые табом.

0.1.4 beta	Добавлена возможность вывода гамма-постоянной по энергиям и расчёта дозы

0.1.0 beta	первая версия программы позволяющая конвертировать библиотеки из одного формата в другой (lib <-> txt <-> tlb). Проводить сортировки и объединение нуклидов в один.
