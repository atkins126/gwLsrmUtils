Программа Spe2TxtConverter предназначена для конвертации между разными форматами файлов спектров (пока есть только spe, txt и sps (открытие)).
Формат командной строки: Spe2TxtConv.exe <InFileName> <outFileExtension or OutFileName> [-d]
InFileName -- имя исходного файла для конвертирования
outFileExt -- формат в который будет проведено конвертирование, если этот формат совпадает с форматом входного файла, то будет создан файл с расширением new
OutFileName -- можно указать имя файла, в который будет произведено конвертирование
-d -- ключ для сохранение txt-формате в урезанном виде, только дата и время набора, живое и реальные времена, число каналов

Примеры: 
1) конвертация Cs137-N500-point-10cm-1.spe в txt формат:
Spe2TxtConv.exe Cs137-N500-point-10cm-1.spe txt
2) массовая конвертация всех текстовых файлов в папке в spe-формат:
for %%i in (*.txt) do Spe2TxtConv.exe %%i spe
