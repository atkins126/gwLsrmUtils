Программа NuclideCalc преданазначена для пересчёта активности нуклида с одной даты на другую
Возможности программы:
- кроме библиотеки по умолчанию, можно использовать свои библиотеки
- если при запуске программы в буфере находилось название нуклида, то программа находит этот нуклид в списке (если он есть).
Кроме имени нуклида там может находиться и старая активность и дата, на которую она приведена. Информация должна быть в виде NucName A=activity ActUnits dA=ActError% Date. Например: Eu-152 A=46880 Бк dA=3% 10-09-2012
Формула для пересчёта активности: Anew=Aold*2^(-(newdate-olddate)/365.25/T_1/2)

Программа считывает данные по периодам полураспада нуклидов из файла NuclideCalc.lib в папке с программой. Самый простой способ добавить нуклид -- отредактировать этот файл. Данные в нём хранятся в виде нуклид символ табулятора период полураспада в годах (десятичный разделитель -- '.').

nuclidecalcc -- консольная версия программы.
формат ввода: 	<Nuclide> <oldAct> <oldDate> <newDate>
		<tHalfLife> <oldAct> <oldDate> <newDate>
Выводит новую активность
