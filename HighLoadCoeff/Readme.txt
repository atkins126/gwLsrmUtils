HighLoadCoeff позволяет рассчитать коэффициенты высокой загрузки для SpectraLine из скоростей счёта, измеренных по методике в руководстве SL.
n - скорости счёта в пике полного поглощения,
r - загрузка (сумма отсчётов в спектре, делённая на живое время)
p - Сумма отсчётов в каждом канале, умноженного на номер канала. Сумма нормирована на живое время.
