Программа для чтения информации о зонах из txt-файлов, полученных из SL (ctrl+z, save data to text file) либо из буфера обмена (copy to clipboard)
Open/Paste - загрузка из файла/буфера обмена информации о зонах
Clear results - очистка
Tools-> calc ratio - считает отношения площадей зон по отношению к выделенной зоне либо к сумме площадей в зависимости от отмеченного Abs ratio
Options-> Abs ratio - меняет способ расчёта отношений площадей зон. При установленной галке - считает по отношению к сумме площадей зон, при снятой -- по отношению к выделенной зоне
ПКМ на списке зон можно указать нуклид для каждой зоны, эти нуклиды будут использоваться для вывода результатов
