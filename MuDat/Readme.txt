MuDat -- программа для вывода массовых коэффициентов поглощения из БД NIST XCOM (http://www.nist.gov/pml/data/xcom/index.cfm)

Позволяет вывести массовые коэффициенты поглощения (Mass attenuation coefficient) для элементов периодической таблицы химических элементов Менделеева.
Можно задавать порядковый номер элемента (Z) либо его обозначение в таблице. Например, водород Z=1, обозначение - H.
Значения энергии, для которых выводить коэфф. поглощения, можно задавать вручную в текстовом поле слева либо задавать сеткой. Сетку можно установить, задав начальное и конечное значения энергий, число точек в сетки и логарифмическую или нет использовать. Установленную сетку можно сохранить, либо загрузить из файла.
Значения массовых коэффициентов поглощения для заданных энергий выводятся по кнопке "Show mju" в правом текстовом поле
Результаты можно сохранить в текстовый файл
