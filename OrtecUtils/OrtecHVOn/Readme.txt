OrtecHVOn.exe
Программа предназначена для поднятия высокого напряжения на спектрометрах Ortec типа DSPec.
Предварительная настройка программы: список доступных трактов должен быть сформирован в MCB Configuration фирмы Ortec. Тогда эти тракты появятся в выпадающем списке OrtecHVOn. При подключении новых трактов необходимо их заново переконфигурировать.
Работа с программой.
Выбрать в выпадающем списке нужный тракт. Подключение произойдёт автоматически при выборе этого тракта. При следующем запуске программа автоматически подключится к последнему выбранному тракту.
После подключения программа покажет состояние высоковольтника спектрометра: значение высокого напряжения, до которого программа будет поднимать напряжение (Target), текущее значение высокого (Actual), метод сброса высокого напряжения (Shutdown method), поднято либо опущено высокое (on или off в левой панели внизу программы) и нет ли перегрузки (в правой панели). Все значения носят информативный характер -- их изменения не приведут к изменению этих значений спектрометра (из соображений безопасности). Все необходимые изменения параметров нужно проводить в ПО Maestro.
Программа позволяет поднимать и опускать высокое напряжение кнопками On и Off, соответственно.

Запуск из коммандной строки.
Программа позволяет снять высокое при запуске из командной строки с ключом '-S'.
OrtecHVOn.exe [-S [LOW_HV]]
Параметр LOW_HV задаёт значение высокого напряжения, при достижении которого высоковольтником, программа завершает свою работу. По-умолчанию задан 40В, если не нужно, чтобы программа выключалась, то нужно задать значение <=0.
