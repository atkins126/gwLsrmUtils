# AlphaMC

**AlphaMC** -- программа для расчёта эффективности регистрации альфа-частиц спектрометром методом Монте-Карло. Программа использует для расчёта диаметр источника, диаметр детектора и расстояние источник-детектор. На выходе получается эффективность регистрации в данной геометрии.
Для учёта отличий модельной системы от реальной, можно проводить расчёт в 2-х вариантах:

1. Расчёт отношения эффективностей в 2-х геометриях `k=Eff2t/Eff1t`, для вычисления реальной эффективности 2-й геометрии нужно умножить это отношение на реальную эффективность 1-й геометрии: `Eff2=k*Eff1`
2. Расчёт эффективности детектора -- отношение числа зарегистрированных частиц к числу попавших на этот детектор. Эта величина не зависит от геометрии измерения. Для этого нужно рассчитать теор. эффективность в геом. 1 и ввести экспериментальную эффективность в этой же геометрии: "Tools -> Add experimental efficiency". Detector efficiency -- это эффективность детектора и на неё будут домножаться расчётные эффективности и выводится в колонке "Calc with det efficiency". Диаметр и эффективность детектора можно сохранить в свой ini-файл. Загружать этот файл можно из меню, а также, указав имя файла в качестве параметра в командной строке

Выбор количества испытаний: число испытаний нужно выбирать таким образом, чтобы статистическая погрешность расчёта была ниже систематических погрешностей в используемых нами эксп. эффективностях. Достаточно, чтобы стат погрешность была меньше в 2 раза.

Можно учитывать то, что детектор немного утоплен, тогда нужно давать расстояние от источника до края детектора. Либо можно оставить эти параметры равными 0, указывать расстояние от источника до кристалла. Расчёт будет правильным до размеров источника <= диаметр детектора + 2`*`расстояние источник-корпус детектора.
