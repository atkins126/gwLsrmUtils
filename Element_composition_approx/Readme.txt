ElComRhoApprox -- программа, считающая элементный состав для сыпучки (ОИСН), изготовленной на основе эпоксидной смолы с добавлением железа, в зависимости от плотности сыпучки.
Нужно ввести плотность сыпучки в г/см3 и нажать на Calc
Установленный CheckBox "Corect to table values" -- интерполяция по табличным данным элементного состава (по 2-м ближайшим точкам), даёт более точный результат, снятая галка -- аппроксимирующая прямая по всем табличным данным.
Уравнение зависимости: 1/Rho = A*md(Fe) + B*md(Rest), 
где A, B -- коэффициенты и имеют смысл обратных плотностей железа и эпоксидной смолы, соответственно
